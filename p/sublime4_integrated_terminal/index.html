Terminus

https://github.com/randy3k/Terminus#user-key-bindings

  1. 安装插件Terminus

  2. 配置字体:

    2.1. Preferences-->setting

2
3
4

    {
  "font_face": "Sarasa Mono SC Nerd",
  "panel_background_color": "$background_color",
}

    2.2 Preferences-->setting-->Package Setgings-->Terminus-Settings

    1

3
4

    {	
  "view_settings": {
    "font_face": "Sarasa Mono SC Nerd",
  },
}

  3. 配置快捷键:

    Preferences-->setting-->Package Setgings-->Terminus-Settings

    1
2
3
4

    [
  { 
      "keys": ["ctrl+`"], "command": "toggle_terminus_panel", "args": {
          "cmd": "/bin/zsh",
          "cwd": "${file_path:${folder}}"
      }
  }
]

注意事项

  • Sublime Text >= 4114 and Terminus v0.3.20 解决内存占用问题.

  • 关闭可以用exit

  • 设置nerd字体解决符号异常
