bios设置

豆子峡谷（NUC8ixBEx）的具体设置项如下：（启用为勾选，禁用为不勾选)

  • 开机时，连续点按 F2 进入 BIOS，为了避免之前有其他不合适的改动，建议先按 F9 重置 BIOS 默认设置。
  • Boot -> Boot Priority -> Legacy Boot Priority -> Legacy Boot ：禁用
  • Boot -> Boot Configuration ->
    • UEFI Boot -> Fast Boot ： 禁用
    • UEFI Boot -> Boot USB Devices First ： 启用
    • UEFI Boot -> Boot Network Devices Last ：启用
    • Boot Devices -> Network Boot ：设置为 Disable
    • Boot -> Secure Boot -> Secure Boot ：禁用
  • Security -> Security Features -> Inter VT for directed I/VO (VT-d) ： 禁用
  • Power -> Secondary Power Settings -> Wake on LAN from S4/S5 ： 设置为 Stay Off

以下为使用白果拆机卡的用户设置：

  • Devices->Onboard Devices-> WLAN 和 Bluetooth » ：禁用

其他网友推荐的bios设置

  • Devices -> USB -> Port Device Charging Mode: off
  • Devices -> USB -> USB Legacy -> Disabled
  • Security -> Thunderbolt Security Level: Legacy Mode

可选:

  • 修改硬盘灯
  • 修改风扇转速
  • 限制功率

手动迁移三码

Hackintool 生成随机三码

https://checkcoverage.apple.com 检查是否有效(We’re sorry, but this serial number isn’t valid. 说明没有人使用过, 有效)

MacSerialTransporter 快速迁移三码

如果要手动迁移三码，请确保以下信息正确：

  • PlatformInfo -> Generic -> MLB xxxxxxxxxxxxxxxxx
  • PlatformInfo -> Generic -> ROM 000000000000
  • PlatformInfo -> Generic -> SystemSerialNumber xxxxxxxxxxxx
  • PlatformInfo -> Generic -> SystemUUID 00000000-1111-2222-3333-444444444444

机型伪装:

  • SystemProductName Macmini8,1

  • ROM 值是您的以太网 (en0) mac 地址

重建缓存并修复权限

打开 Hackintool 软件，点击顶部“工具”页面，点击最下面一排图标的最后一个（鼠标悬浮在上面显示“重建缓存并修复权限”的便是）。此操作大约需要等待半分钟执行完毕。

如何清空NVRAM

电脑启动时，连续按 ESC 键进入 OC 引导界面，按一次空格键出现隐藏选项，选择最后一个“Reset NVRAM”回车执行，电脑会自动重启进入系统。

会丢失uefi启动项,需要使用pe重新添加,只有在必要的时候才用

完美的休眠

我们需要修改Hackintool里的电源配置，按切换到“电源”页面，点击“修复深度休眠预留空间”，输入登录密码并确认后，保证 hibernatemode、proximitywake 均为绿色

下面的设置前可以pmset -g查看当前状态

sudo pmset standby 0 # 禁用待机
sudo pmset autopoweroff 0 # 禁用自动关机
sudo pmset proximitywake 0 # 禁用使用iPhone、iWatch等设备唤醒
sudo pmset powernap 0 # 禁用电源小憩
sudo pmset tcpkeepalive 0 # 禁用TCP keepalive
sudo pmset womp 0 # 禁用休眠唤醒
sudo pmset hibernatemode 0 # 禁用hibernate

trim

检查是否开启:

苹果图标 －> 关于本机 －> 概览，点击『系统报告』，弹出新窗口，在左边硬件目录下，点击『SATA/SATA Express』，然后在右边会看到“串行ATA设备树”，在固态硬盘里找到TRIM支持，如果显示是表示TRIM已经开启。

运行sudo trimforce enable启用

单系统config改双硬盘双系统

修改config.plist

  • AllowToggleSip: 改成 true , 作用是在op界面增加快速切换sip选项
  • ShowPicker: 改成 true, 硬盘切换选择项
  • misc->boot->timeout: 改成 5 , 作用是选择硬盘等待时间
